# DOXA Detector

DOXA Detector is an experimental cognitive calibration tool designed to evaluate the structural robustness and credibility of written discourse.

The system is based on a simple cognitive heuristic:

M = (G + N) − D

G = Gnōsis — articulated and verifiable knowledge  
N = Nous — coherent reasoning and logical integration  
D = Doxa — stabilized opinion or unsupported assertion

The model estimates the cognitive robustness of a text by balancing knowledge, reasoning, and opinion.

## Purpose

DOXA Detector analyzes articles, texts, and claims to distinguish structurally robust information from fragile or unsupported discourse.

Rather than simply classifying statements as true or false, the system evaluates the **cognitive structure of the argument itself**.

## Main Features

• Text analysis  
• Article analysis via URL  
• Multi-article topic analysis  
• Hard Fact Score (credibility score)  
• Cognitive diagnostic indicators  
• AI-assisted critique  

## Hard Fact Score

The application produces a credibility score estimating the structural reliability of a text.

Typical interpretation:

0–5   : very fragile  
6–10  : uncertain  
11–15 : solid  
16–20 : structurally robust
